# vue-with-jwt-athentication

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve 
```

### Compiles and minifies for production
```
yarn build
```

### running server 
```
go to build dir 
yarn start
```

### Lints and fixes files
